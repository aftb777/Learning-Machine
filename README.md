# Learning-Machine 🚀

A complete collection of **Machine Learning algorithms implemented from scratch** for learning and experimentation.

---

## 📌 Overview

This repository focuses on understanding **core Machine Learning concepts deeply** by implementing algorithms manually instead of relying on high-level libraries.

---

## ⚙️ Tech Stack

* Python 3.x
* NumPy
* Pandas
* Matplotlib / Seaborn
* Jupyter Notebook
* Scikit-learn (for validation only)

---

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```
git clone https://github.com/aftb777/Learning-Machine.git
cd Learning-Machine
```

### 2️⃣ Install Dependencies

```
pip install numpy pandas matplotlib seaborn jupyter scikit-learn
```

### 3️⃣ Run Notebook

```
jupyter notebook
```
